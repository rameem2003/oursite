NetBeans ও JDK এক নজরে

নেটবিন্স একটি জাভা ডেস্কটপ অ্যাপ্লিকেশন তৈরির প্লাটফর্ম। এছাড়া নেটবিন্স দিয়ে জাভাস্ক্রিপ্ট, পিএইচপি, গ্রুভি, সি,সি++ ইত্যাদি ভাষা নিয়ে কাজ করা যায়।নেটবিন্স জাভা দিয়ে তৈরি করা এবং উইন্ডোজ,লিনাক্স, ম্যাকসহ প্রায় সকল অপারেটিং সিস্টেমে কাজ করে। নেটবিন্স ব্যবহারকারীকে বিভিন্ন মডিউল তৈরি করে সফটওয়্যার বানানোর সুবিধা দেয়। নেটবিন্স দিয়ে তৈরি সফটওয়্যার পরবর্তীকালে অন্য ডেভেলপাররা ডেভেলপ করতে পারে। এছাড়া নেটবিন্স দিয়ে সহজে ড্রাগ-ড্রপ করে গ্রাফিকাল ইন্টারফেস বানানো যায়।

জাভা ডেভেলপমেন্ট কিট (JDK) হল ওরাকল কর্পোরেশনের জাভা প্রযুক্তির একটি বিতরণ। এটি জাভা ল্যাঙ্গুয়েজ স্পেসিফিকেশন (JLS) এবং জাভা ভার্চুয়াল মেশিন স্পেসিফিকেশন (JVMS) প্রয়োগ করে এবং জাভা অ্যাপ্লিকেশন প্রোগ্রামিং ইন্টারফেস (API) এর স্ট্যান্ডার্ড সংস্করণ (SE) প্রদান করে। এটি সম্প্রদায় চালিত OpenJDK এর ডেরিভেটিভ যা ওরাকল স্টুয়ার্ড। এটি জাভা অ্যাপ্লিকেশনগুলির সাথে কাজ করার জন্য সফ্টওয়্যার সরবরাহ করে। অন্তর্ভুক্ত সফ্টওয়্যারগুলির উদাহরণ হল ভার্চুয়াল মেশিন, একটি কম্পাইলার, কর্মক্ষমতা পর্যবেক্ষণ সরঞ্জাম, একটি ডিবাগার এবং অন্যান্য ইউটিলিটি যা ওরাকল একটি জাভা প্রোগ্রামারের জন্য দরকারী বলে মনে করে।

JDK ডাউনলোড ও ইন্সটল করার পদ্ধতি

১) JDK ডাউনলোড করার জন্য প্রথমে আপনার পছন্দের ওয়েব ব্রাউজার এ গিয়ে এই লিঙ্কটি https://www.oracle.com/java/technologies/downloads/ বা সরাসরি এই লিংকে JDK download ক্লিক করুন। এরপরে Oracle এর অফিসিয়াল ওয়েবসাইট টি চলে আসবে।

২) এরপরে নিচে স্ক্রল করে Windows সিলেক্ট করতে হবে। এরপরে https://download.oracle.com/java/18/latest/jdk-18_windows-x64_bin.exe তে ক্লিক করতে হবে। JDK ডাউনলোড হওয়া শুরু হয়ে যাবে।

৩) এরপরে Setup file টির ওপর ডাবল ক্লিক করলে সেটআপ ইন্টারফেস টি সামনে আসবে।

৪) এরপরে Next ক্লিক করতে হবে।

৫) এরপরে আবার Next ক্লিক করতে হবে।

৬) কিছুক্ষন পরে JDK ইন্সটল হয়ে যাবে।

NetBeans IDE ডাউনলোড ও ইন্সটল করার পদ্ধতি

১) NetBeans IDE ডাউনলোড করার জন্য প্রথমে আপনার পছন্দের ওয়েব ব্রাউজার এ গিয়ে এই লিঙ্কটি https://netbeans.apache.org/download/nb13/nb13.html বা সরাসরি এই লিংকে NetBeans IDE download ক্লিক করুন। এরপরে NetBeans এর অফিসিয়াল ওয়েবসাইট টি চলে আসবে। এরপরে Apache-NetBeans-13-bin-windows-x64.exe তে ক্লিক করতে হবে।

২) এরপরে একটি পেজ আসবে সেখানে মার্ক করা প্রথম লিংকটি ক্লিক করলে সফটওয়্যার টি ডাউনলোড হওয়া শুরু করবে।

৩) এরপরে Setup file টি ডাবল ক্লিক করলে সেটআপ ইন্টারফেস টি চলে আসবে

৪) এরপরে Next এ ক্লিক করতে হবে।

৫) এরপরে লাইসেন্স এ টিক দিয়ে Next এ ক্লিক করতে হবে।

৬) এরপরে আবার Next ক্লিক করতে হবে।

৭) এরপরে Next ক্লিক করলে সফটওয়্যারটি ইন্সটল হওয়া শুরু করবে।

৮) Finish ক্লিক করলে, অতঃপর সফটওয়্যারটি অপেন হবে।
