Virtual Box এক নজরে

ওরাকল ভিএম ভার্চুয়ালবক্স (পূর্বে সান ভার্চুয়ালবক্স , সান এক্সভিএম ভার্চুয়ালবক্স এবং ইনোটেক ভার্চুয়ালবক্স ) ওরাকল কর্পোরেশন দ্বারা তৈরি x86 ভার্চুয়ালাইজেশনের জন্য একটি টাইপ-2 হাইপারভাইজার । ভার্চুয়ালবক্স মূলত ইনোটেক জিএমবিএইচ দ্বারা তৈরি করা হয়েছিল, যা 2008 সালে সান মাইক্রোসিস্টেম দ্বারা অধিগ্রহণ করা হয়েছিল , যা 2010 সালে ওরাকল দ্বারা অধিগ্রহণ করা হয়েছিল। ভার্চুয়ালবক্স Microsoft Windows , macOS , Linux , Solaris এবং OpenSolaris- এ ইনস্টল করা হতে পারে । এছাড়াও ফ্রিবিএসডি এবং জেনোডের পোর্ট রয়েছে । এটি উইন্ডোজ, লিনাক্স, বিএসডি , ওএস/২ , সোলারিস, হাইকু এবং ওএসএক্স৮৬ , চালিত গেস্ট ভার্চুয়াল মেশিন তৈরি ও পরিচালনার পাশাপাশি অ্যাপল হার্ডওয়্যারে ম্যাকোস গেস্টদের সীমিত ভার্চুয়ালাইজেশন সমর্থন করে।

Virtual Box ডাউনলোড ও ইন্সটল করার পদ্ধতি

১) Virtual Box ডাউনলোড করার জন্য প্রথমে আপনার পছন্দের ওয়েব ব্রাউজার এ গিয়ে সরাসরি এই লিংকে https://www.virtualbox.org/ ক্লিক করুন। এরপরে ভার্চুয়াল বক্স এর অফিসিয়াল ওয়েবসাইট টি চলে আসবে। এরপরে Download তে ক্লিক করতে হবে।

২) এরপরে Download for windows host এ ক্লিক করলে ভার্চুয়াল বক্স এর সেটআপ ফাইলটি ডাউনলোড হয়ে যাবে।

৩) এরপরে সেটআপ ফাইলের ওপর ডবল ক্লিক করলে Installation interface টি চলে আসবে

৪) Next এ ক্লিক করতে হবে।

৫) Next এ ক্লিক করতে হবে।

৬) Yes এ ক্লিক করতে হবে।

৭) Install এ ক্লিক করতে হবে।

৮) কিছুক্ষণ এর মধ্যে সফটওয়্যার টি Install হয়ে যাবে।

Ubuntu ডাউনলোড করার পদ্ধতি

১) Ubuntu ডাউনলোড করার জন্য প্রথমে আপনার পছন্দের ওয়েব ব্রাউজার এ গিয়ে সরাসরি এই লিংকে https://ubuntu.com/ ক্লিক করুন। এরপরে Ubuntu এর অফিসিয়াল ওয়েবসাইট টি চলে আসবে। এরপরে Download তে ক্লিক করতে হবে।

২) এরপরে Download 22.04 LTS এ ক্লিক করলে Ubuntu এর ISO ফাইলটি ডাউনলোড হয়ে যাবে।

Virtual box এ Ubuntu ইন্সটল করার পদ্ধতি

১) প্রথমে Virtual box software টি অপেন করে নিতে হবে।

২) এবার New তে ক্লিক করতে হবে। New তে ক্লিক করার পর অপারেটিং সিস্টেম এর একটি নাম দিতে হবে। যেমন এখানে Ubuntu নাম দেওয়া হল। এরপরে OS টি কোথায় save হবে সেই লোকেশন দেখিয়ে দিতে হবে। এবার Next এ ক্লিক করতেহ হবে।

৩) এবার আমাদের OS টির জন্য Ram সাইজ নির্ধারণ করতে হবে। এক্ষেত্রে আমাদের মুল সিস্টেমে যতটুকু Ram আছে, তার অর্ধেক নির্ধারণ করা উত্তম। এবার Next এ ক্লিক করতেহ হবে।

৪) এবার Create এ ক্লিক করতে হবে।

৫) এরপরে Create virtual hard disk সিলেক্ট করে Next এ ক্লিক করতে হবে।

৬) Next এ ক্লিক করে আমাদের OS টির জন্য স্টোরেজ সাইজ নির্ধারণ করতে হবে। এরপরে Create এ ক্লিক করলে আমাদের ভার্চুয়াল মেশিন তৈরি হয়ে যাবে।

৭) Start এ ক্লিক করলে ভার্চুয়াল মেশিনটি চালু হবে এবং একটি পপআপ স্ক্রিন আসবে। সেখানে আগে থেকে ডাউনলোড করে রাখে Ubuntu ISO file টি লোকেশন দেখিয়ে দিতে হবে। এরপরে Start এ ক্লিক করলে আমাদের ভার্চুয়াল মেশিনে OS টি ইন্সটল শুরু হবে।

৮) এরপরে Ubuntu এর Welcome স্ক্রিন টি আসবে।

৯) কিছুক্ষণ পরে ইন্সটললেশন Window টি চলে আসবে। Install Ubuntu তে ক্লিক করতে হবে।

১০) কি বোর্ড এর ভাষা English select করে Next তে ক্লিক করতে হবে। এরপরে Continue ক্লিক করতে হবে।

১১) Continue ক্লিক করতে হবে।

১২) Install now ক্লিক করতে হবে।

১৩) এবার Username, password ইত্যাদি সেট করে Continue তে ক্লিক করলে install শুরু হয়ে যাবে।

১৪) কিছুক্ষণ পরে Ubuntu এর ডেক্সটপ স্ক্রিন টি চলে আসবে।
