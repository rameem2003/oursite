Git এক নজরে

গিট হল একটি ডিস্ট্রিবিউটেড সংস্করণ নিয়ন্ত্রণ পদ্ধতি যা প্রধানত প্রোগ্রামিং প্রকল্পের উৎস কোডের পরিবর্তন সংরক্ষণে ব্যবহার করা হয়। এটি প্রোগ্রামারদের মধ্যে কাজের সমন্বয় করার উদ্দেশ্যে তৈরি হলেও যেকোন ধরনের কম্পিউটার ফাইলের পরিবর্তন রক্ষায় ব্যবহার করায় কোন বাধা নেই।

Git ডাউনলোড ও ইন্সটল করার পদ্ধতি

১) Git ডাউনলোড করার জন্য প্রথমে আপনার পছন্দের ওয়েব ব্রাউজার এ গিয়ে এই লিংকে https://git-scm.com/ প্রবেশ করুন। এরপরে Git SCM এর ওয়েবসাইট টি চলে আসবে। এরপরে Download for windows তে ক্লিক করলে একটি ছোট ফাইল ডাউনলোড জয়ে যাবে।

২) এরপরে ফাইলটি ডবল ক্লিক করে ওপেন করতে হবে। সেটআপ ফাইলটি আসলে লাইসেন্স accept করে NEXT এ ক্লিক করতে হবে। এরপরে স্বাভাবিক সফটওয়্যার ইন্সটল করার মত NEXT, NEXT ক্লিক করে। সফটওয়্যার টি ইন্সটল করতে হবে

Git setup করার পদ্ধতি

১) Git install এরপরে Start menu থেকে Git Bash সার্চ করে চালু করতে হবে। চালু করলে এমন একটি স্ক্রিন আসবে।

৩) এই পর্যায়ে Git Bash এ দুটি কমান্ড দিতে হবে

git config --global user.name "your git username"

git config --global user.email "your git email"

Tarminal দিয়ে আপলোড করার পদ্ধতি

৪) এরপরে Visual Studio Code এর একটি প্রজেক্ট ফোল্ডার ওপেন করে VS Code এর Tarminal চালু করে ফোল্ডারটিকে ইনিশিয়াল করতে হবে।(এর আগে আমাদের একটি Github account থাকতে হবে এবং একটি রিপজিটরি বানাতে হবে) ইনিশিয়াল করার কোড গুলোঃ

git init

git add .

git commit -m "massage"

git remote add origin "git link"

git push -u origin main

VS Code Source Control দিয়ে আপলোড করার পদ্ধতি

১) VS Code এর Git Control অপশন এ গিয়ে Commit section এ গিয়ে টিক মার্কে ক্লিক করতে হবে।
